## Установка

  - Скачиваем проект
  - Переходим в папку проекта
  - Устанавливам зависимости командой `npm install`
  - Запускаем проект командой `npm start`
  - Открываем в браузере адрес [http://localhost:8080](http://localhost:8080)

## Используемые пакеты
